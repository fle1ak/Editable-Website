Index.html и Style.css

index.html — это стандартное имя главного HTML-файла веб-сайта. Когда пользователь переходит
на доменное имя или папку веб-сайта (например, https://example.com/), веб-сервер по умолчанию ищет
файл с именем index.html, чтобы отобразить его как главную страницу сайта.


CSS (Cascading Style Sheets) — это язык описания внешнего вида (стилей) веб-страниц.
С помощью CSS можно изменять оформление элементов HTML-документа, таких как цвета, шрифты,
отступы, размеры, расположение и другие визуальные параметры. Это позволяет разделить структуру
(HTML) и стиль (CSS), что упрощает разработку и поддержку сайтов.
